This project is a Math Visualizer created for Grade 12 students to develop a greater understanding of vectors and vector operations, with the use of drag and drop features and visual components to help students understand vector concepts. In this project, I primarily worked on the Vector Addition and Dot Product question pages. Furthermore, I developed the template that was used for the Vector Subtraction and Dot Product question pages. This includes the feature that allows the vectors to be dragged and placed in the correct position, as well to receive user input for the user's answer and provide them with the necessary feedback.
