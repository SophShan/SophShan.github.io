About

I am an undergraduate student pursuing a bachelor's computer science at McMaster University. Some skills of mine include programming in Java, Python, C, Elm, and Haskell. I also have experience in Bash Shell Scripting as well as writing queries with SQlite3 with Python. Additonally, I have worked with file input and output with JSON, CSV and Yaml files using Python.

The above first two projects showcase some of the work that I have created in group environments. The first project listed is a website that provides students from kindergarten to grade 5 with randomized math questions from four categories, and times how long it takes for them to complete ten questions of the type they have selected. The second project is a recreation of the mini games in the online game, Amoung Us. My primary task in this project was to recreate the Amoung Us Leaf Clean Up task, which is located in the reactor room of the game. Users can travel through different parts of the map by clicking on the room they desire to go to. This map was intentionally designed so that the user can not revisit previous main rooms of the map without making a full circle of the map. The third project is a recreation of game Connect Four and was a personal school project I took on. This website is also one of my own projects, and uses html and css.
